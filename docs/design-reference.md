# Design Reference

## 1) 문서 목적
- 이 문서는 홍천 꽃신 랜딩 페이지의 UI/UX 구현 기준입니다.
- `꽃신`은 한국 전통 신발이 아니라 **꽃피는 신장대리**의 줄임말입니다.
- 방문자가 이름을 오해할 수 있다는 점은 브랜드의 작은 유머 포인트로 사용하되, 디자인의 중심은 실제 공간 사진과 현대적인 마을 카페 분위기에 둡니다.

## 2) 핵심 방향성
- 실제 제공 사진 기반의 이미지 중심 경험
- 밝은 창, 나무 질감, 식물, 회색 바닥, 화이트 사인을 살린 현대적인 카페 톤
- 전통 연출보다 `동네성`, `편안함`, `깔끔함`, `방문하기 쉬움` 강조
- 예약 CTA는 어디서든 명확하게 인지 가능해야 함

## 3) 반드시 지킬 원칙
- 한 화면(섹션)당 핵심 메시지는 1개만 전달
- 이미지는 장식이 아니라 실제 공간/메뉴/위치 정보로 사용
- 이름 설명은 짧고 기억에 남게: `꽃신은 꽃피는 신장대리의 줄임말입니다`
- 전통 꽃신 이미지는 오해를 설명하는 보조 장면으로만 사용
- 과도한 패턴, 민속적 장식, 한지/문양 중심 연출은 사용하지 않기

## 4) 정보 구조(랜딩 기준)
1. Hero: 실제 실내 공간 사진 + 이름의 뜻 + 예약 CTA
2. Brand Story: 꽃피는 신장대리 이름 설명과 작은 유머 포인트
3. Space Gallery: 실제 내부/쇼케이스/전시 사진
4. Menu: 실제 메뉴판 사진 + 텍스트 메뉴 목록
5. Reservation CTA: 매장/포장 예약 요청
6. Location & Contact: 위치 사진 + 연락 정보
7. Footer: 브랜드 한 줄 설명

## 5) 비주얼 시스템 가이드
- Background: `#F6F7F4` 계열의 밝은 오프화이트
- Text: 차콜/블랙에 가까운 안정적인 색
- Accent: 식물과 어울리는 딥그린
- Support: 우드톤은 사진에서 자연스럽게 드러내고 CSS 색으로 과하게 덮지 않기
- Radius: 카드/이미지는 8px 내외로 단정하게

## 6) 이미지 사용 기준
- Hero 우선 후보: `hongcheon-kkotsin-11.jpg`
- 공간 갤러리: `05`, `03`, `13`, `11`
- 메뉴판: `hongcheon-kkotsin-08.png`
- 특별 메뉴: `09`, `10`
- 위치/간판: `01`, `02`
- 층별 안내: `12`

## 7) 카피 톤
- 설명은 짧고 친근하게 쓴다.
- “전통 꽃신”을 메인 콘셉트로 밀지 않는다.
- 오해 포인트는 가볍게 인정한다.
  - 예: `전통 꽃신인 줄 알았다면, 정상입니다.`
  - 예: `홍천 꽃신의 꽃신은 꽃피는 신장대리의 줄임말입니다.`

## 8) 구현 체크리스트
- [ ] 첫 화면에서 실제 공간 사진이 바로 보이는가
- [ ] 꽃신의 뜻이 명확히 설명되는가
- [ ] 전통 신발/전통 무드 중심으로 오해되지 않는가
- [ ] 메뉴판 사진과 텍스트 메뉴가 함께 제공되는가
- [ ] 예약 CTA가 상단/중단에서 명확한가
- [ ] 모바일에서 이미지와 버튼이 겹치지 않는가
