# Design Reference

## 1) 문서 목적
- 이 문서는 한국 전통차 카페 랜딩 페이지의 UI/UX 구현 기준입니다.
- 목표는 `텍스트 중심`이 아닌 `이미지 중심` 경험을 만드는 것입니다.
- 참고 템플릿(Wix 3096)은 **구조/밀도/분위기 레퍼런스만 참고**하며, 직접 복제하지 않습니다.

## 2) 핵심 방향성
- 높은 이미지 밀도, 낮은 텍스트 밀도
- 한국의 아름다운 풍경(특히 홍천 자연 풍경)을 은은한 배경으로 활용
- 차분하고 고급스럽고 따뜻한 분위기
- 세련된 한국 전통 무드(현대적 해석)
- 예약 CTA는 어디서든 명확하게 인지 가능해야 함

## 3) 반드시 지킬 원칙
- 한 화면(섹션)당 핵심 메시지는 1개만 전달
- 텍스트는 설명보다 분위기/행동 유도에 집중
- 이미지는 `장식`이 아니라 `콘텐츠`로 사용
- 전통 요소는 직접적 상징(민속촌 느낌)보다 재질감/색/여백으로 표현
- 과도한 장식 패턴, 강한 대비, 과한 채도 사용 금지

## 4) 피해야 할 톤
- 관광지 기념품샵 같은 전통 연출
- 한지 패턴/문양의 과잉 사용
- 많은 문단과 긴 설명문 위주의 첫 화면
- 배경 이미지 위 저가독성 텍스트 배치

## 5) 정보 구조(랜딩 기준)
1. Hero (첫 화면)
2. Tea & Space Visual Story (이미지 중심 소개)
3. Signature Tea / Seasonal Picks (카드형 하이라이트)
4. Reservation CTA Block (강조)
5. Location & Atmosphere (풍경 + 지도/오시는 길 요약)
6. Footer (운영 정보 최소 표기)

## 6) 섹션별 UX 가이드

### Hero
- 풀블리드 배경 이미지(홍천 풍경 또는 카페 공간) 사용
- 카피는 1~2문장 이내
- CTA는 `예약하기` 단일 우선 버튼 + 보조 버튼 1개 이내
- 스크롤 유도 요소는 미세하게 표현

### Tea & Space Visual Story
- 2~6장 이미지 그리드/모자이크 중심
- 텍스트는 이미지 캡션 수준으로 최소화
- 이미지 간 여백으로 고급스러운 리듬 형성

### Signature Tea / Seasonal Picks
- 카드 수 3~4개 권장
- 각 카드: 이미지 > 이름 > 매우 짧은 설명(1줄)
- 상세 정보는 랜딩에서 축약하고, 필요 시 별도 페이지로 분리

### Reservation CTA Block
- 배경 대비가 높은 단일 블록으로 배치
- 문구는 짧고 명확하게: 예) `조용한 시간, 차 한 잔의 여유를 예약하세요`
- 버튼은 강한 대비 컬러와 충분한 클릭 영역 사용

### Location & Atmosphere
- 지도 텍스트를 장문으로 쓰지 말고 핵심 정보만 제공
- 풍경 이미지와 함께 접근성(주차/대중교통) 요약 제공

## 7) 비주얼 시스템 가이드

### 컬러 무드(예시)
- Warm Ivory: `#F4EFE6` (배경 기본)
- Tea Brown: `#6B4E3D` (포인트/버튼)
- Ink Charcoal: `#2E2A27` (텍스트)
- Muted Sage: `#8B9A8A` (보조 포인트)
- Mist Beige: `#D9CFC2` (구분면/보조 배경)

### 타이포그래피 방향
- 제목: 한국어 세리프 계열(단정하고 고급스러운 인상)
- 본문/UI: 가독성 높은 산세리프
- 텍스트 크기보다 행간/여백으로 고급감 확보
- 긴 카피보다 짧은 문장 + 강한 시각 흐름 우선

### 이미지/배경 처리
- 풍경 이미지는 저채도/저대비 보정으로 은은하게 처리
- 배경 오버레이(어두운 베이지/브라운)로 텍스트 가독성 확보
- 필요 시 아주 약한 블러/그레인으로 질감 통일
- 인물 중심보다 공간/자연 중심 컷 우선

## 8) 밀도 규칙
- 첫 화면 기준 이미지 점유감이 명확해야 함(비주얼 우선)
- 한 섹션 내 본문 문단은 최대 2~3문장
- 한 화면에 버튼 우선순위는 최대 2개(Primary 1, Secondary 1)
- 스크롤 흐름에서 `이미지 -> 짧은 정보 -> 행동(예약)` 패턴 반복

## 9) 예약 CTA 규칙
- 상단 네비게이션에 `예약하기` 상시 노출
- 주요 구간마다 반복 노출(중단 CTA + 하단 CTA)
- 모바일에서는 하단 고정 CTA 또는 플로팅 버튼 적용
- 버튼 라벨은 동사형으로 명확하게 유지 (`예약하기`, `지금 예약`)

## 10) 인터랙션 가이드
- 애니메이션은 느리고 절제된 페이드/슬라이드 중심
- 과한 패럴랙스/과도한 마이크로 인터랙션 금지
- 호버/포커스 상태는 색 변화보다 명도 차이로 우아하게 처리

## 11) 접근성/실사용성
- 텍스트 대비 기준 준수(배경 이미지 위 텍스트 특히 주의)
- 버튼 최소 터치 영역 확보
- 핵심 정보(영업시간/위치/예약)는 이미지에만 의존하지 않기
- 모바일에서 배경 이미지 크롭 시 주요 피사체 손실 방지

## 12) 구현 체크리스트
- [ ] Hero가 이미지 중심으로 시작되는가
- [ ] 텍스트가 과밀하지 않은가(장문 제거)
- [ ] 한국 전통 무드가 현대적으로 표현되는가
- [ ] 촌스러운 전통 연출 요소가 배제되었는가
- [ ] 예약 CTA가 모든 주요 구간에서 명확한가
- [ ] 모바일에서도 동일한 분위기와 예약 동선이 유지되는가
